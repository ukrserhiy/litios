# LITI (ЛІТІ) — Психолінгвістичний HR-аналіз

**LITI** - психолінгвістична діагностика для рекрутингу ШІ-система для глибинного аналізу особистості на основі мовленнєвих патернів. Створена групою IT-компаній FRACTAL для об'єктивної оцінки потенціалу кандидатів.<br>
**LITI**  — self-hosted інструмент для психолінгвістичного аналізу транскриптів інтерв'ю. Визначає особистісні риси кандидатів на основі мовних маркерів, використовуючи AI моделі через [OpenRouter](https://openrouter.ai/).

**Особливості:**
- Всі дані зберігаються на вашому сервері (SQLite)
- Історія аналізів не втрачається при перезавантаженні
- Повний контроль над вашими даними
- Один користувач, без авторизації

## Можливості

- **20+ психологічних шкал**: Big Five, HEXACO, Dark Triad, PsyCap та інші
- **Аналіз мовних маркерів**: лексичні, синтаксичні, семантичні та дискурсивні патерни
- **Відповідність вакансії**: аналіз резюме та інтерв'ю на відповідність вимогам
- **Гнучкий вибір моделей**: від безкоштовних (Gemini) до преміум (Claude, GPT)
- **Історія аналізів**: збереження та перегляд попередніх результатів
- **Експорт в PDF**: друк звітів для документації

## Швидкий старт

> **Для новачків:** Якщо ви ніколи не працювали з серверами — читайте [INSTALL_GUIDE.md](INSTALL_GUIDE.md) з покроковими інструкціями.

### 1. Отримайте API ключ OpenRouter

1. Зареєструйтесь на [openrouter.ai](https://openrouter.ai/)
2. Перейдіть в [API Keys](https://openrouter.ai/keys)
3. Створіть новий ключ (назва: `LITI`)
4. Скопіюйте ключ (починається з `sk-or-...`)

### 2. Локальний запуск

#### Варіант A: Docker (рекомендовано)

```bash
# Клонувати репозиторій
git clone https://github.com/ukrserhiy/litios.git
cd litios

# Запустити через Docker Compose
docker-compose up -d

# Відкрити в браузері
# http://localhost:8080
```

#### Варіант B: Python напряму

```bash
# Клонувати репозиторій
git clone https://github.com/ukrserhiy/litios.git
cd litios

# Створити віртуальне середовище
python -m venv venv
source venv/bin/activate  # Linux/Mac
# або: venv\Scripts\activate  # Windows

# Встановити залежності
pip install -r requirements.txt

# Запустити сервер
python server.py

# Відкрити в браузері
# http://localhost:8080
```

### 3. Налаштування

1. Відкрийте додаток в браузері
2. Перейдіть до **Налаштування** (іконка шестерні)
3. Вставте ваш OpenRouter API ключ
4. Виберіть AI модель (рекомендовано: Claude Haiku 4.5)
5. Увімкніть потрібні шкали аналізу

## Використання

### Базовий аналіз

1. Введіть ім'я та прізвище кандидата
2. Вставте текст транскрипту інтерв'ю
3. Натисніть **"Почати аналіз"**
4. Отримайте результати по вибраних шкалах

### Аналіз відповідності вакансії

1. Увімкніть чекбокс **"Вакансія"**
2. Вставте текст вакансії з вимогами
3. (Опціонально) Увімкніть **"Резюме"** та вставте резюме кандидата
4. Запустіть аналіз
5. Отримайте таблицю відповідності вимогам:
   - ✓ — відповідає
   - ✗ — не відповідає
   - ? — не зрозуміло

## Шкали аналізу

### Big Five (Велика п'ятірка)
| Шкала | Опис |
|-------|------|
| Сумлінність | Організованість, відповідальність, планування |
| Екстраверсія | Соціальність, енергійність, комунікабельність |
| Приязність | Співпраця, емпатія, доброзичливість |
| Нейротизм | Емоційна нестабільність, тривожність |
| Відкритість | Креативність, допитливість, новаторство |

### HEXACO
| Шкала | Опис |
|-------|------|
| Чесність-Смиренність | Щирість, скромність, етичність |

### Dark Triad (Темна тріада)
| Шкала | Опис | Інтерпретація |
|-------|------|---------------|
| Нарцисизм | Грандіозність, потреба у визнанні | Високий бал = ризик |
| Макіавеллізм | Маніпулятивність, цинізм | Високий бал = ризик |
| Психопатія | Відсутність емпатії, імпульсивність | Високий бал = ризик |

### PsyCap (Психологічний капітал)
| Шкала | Опис |
|-------|------|
| Надія | Цілеспрямованість, пошук альтернатив |
| Самоефективність | Впевненість у здібностях |
| Стійкість | Відновлення після невдач |
| Оптимізм | Позитивні очікування |

### Інші шкали
- **IPC**: Агентність, Комунальність
- **Regulatory Focus**: Фокус просування, Фокус запобігання
- **SDT**: Автономія, Компетентність, Зв'язаність

## Технічний стек

- **Backend**: Python 3.11, Flask, SQLite
- **Frontend**: Vanilla JS, CSS3 (без фреймворків)
- **AI**: OpenRouter API (Claude, GPT-4, Gemini, DeepSeek)
- **Deployment**: Docker, Gunicorn

## API

### Ендпоінти

| Метод | Шлях | Опис |
|-------|------|------|
| GET | `/api/prompts` | Отримати налаштування промптів і шкал |
| POST | `/api/prompts` | Зберегти промпти |
| GET/POST | `/api/prompts/scales` | Отримати/зберегти шкали |
| GET/POST | `/api/settings` | Налаштування (API ключ, модель) |
| GET/POST | `/api/models` | Отримати/зберегти моделі AI |
| GET | `/api/history` | Історія аналізів |
| POST | `/api/history/add` | Додати аналіз |
| DELETE | `/api/history/{id}` | Видалити аналіз |
| POST | `/api/reset-to-defaults` | Скинути до початкових налаштувань |
